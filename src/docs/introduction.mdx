# Gajozo UI

Biblioteca de componentes React baseada em **copia e cola** — sem dependência de framework de UI, sem pacote npm para instalar.

## O que é?

Gajozo UI é um conjunto de componentes prontos para uso que você copia diretamente para o seu projeto. Cada componente fica em uma pasta isolada com dois arquivos: `index.tsx` e `*.module.css`. Basta copiar a pasta, importar o componente e usar.

## Filosofia

- **Copia e cola:** Abra a story do componente, clique em _Baixar ZIP_ ou copie os arquivos individualmente e cole no seu projeto.
- **Zero dependências de runtime:** Os componentes dependem apenas do React. Nenhuma biblioteca de UI, nenhum utilitário compartilhado.
- **Compatibilidade ampla:** Funciona com React 16.8 (era dos hooks) até React 19 sem nenhuma adaptação. Nenhum uso de `useId`, APIs concorrentes ou RSC.
- **Acessível:** Padrões WAI-ARIA, navegação por teclado e gerenciamento de foco incluídos nativamente.
- **Customizável:** CSS custom properties para sobrescrever tokens de design sem alterar o componente.

## Componentes disponíveis

| Componente      | Descrição                                                                       |
| --------------- | ------------------------------------------------------------------------------- |
| **Select**      | Dropdown de valor único com navegação por teclado, label e campo obrigatório    |
| **MultiSelect** | Dropdown de múltiplos valores com busca, chips, limpar tudo e limite de seleção |

## Como usar um componente

1. Navegue até o componente no menu lateral.
2. Clique em **Copiar index.tsx** ou **Copiar CSS** para copiar o código.
3. Ou clique em **Baixar ZIP** para obter os dois arquivos de uma vez.
4. Cole os arquivos no seu projeto e importe o componente normalmente.

Nenhuma configuração adicional é necessária — cada componente é completamente autossuficiente.
