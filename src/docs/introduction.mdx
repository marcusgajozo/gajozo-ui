# Componentes Reutilizáveis

Biblioteca de componentes React baseada em **copia e cola** — sem dependência de framework de UI, sem pacote npm para instalar.

## O que é?

Este é um conjunto de componentes prontos para uso que você copia diretamente para o seu projeto. Cada componente fica em uma pasta isolada contendo o componente principal, seus subcomponentes e arquivos de estilo `.module.css`. Basta copiar a pasta, instalar a dependência base (se houver, como `@base-ui/react`), importar o componente e usar.

## Filosofia

- **Copia e cola:** Abra a story do componente, clique em _Baixar ZIP_ ou copie os arquivos individualmente e cole no seu projeto.
- **Zero dependências de runtime:** Os componentes dependem apenas do React e de bibliotecas headless utilitárias como o Base UI. Nenhuma biblioteca visual pesada e nenhum utilitário compartilhado que te prenda a um ecossistema.
- **Acessível:** Padrões WAI-ARIA, navegação por teclado e gerenciamento de foco incluídos nativamente (muitos via Base UI).
- **Customizável:** Padrão CSS Modules junto com custom properties para sobrescrever tokens de design sem alterar o componente internamente.
- **Composição:** Usamos o padrão de composição (Composition Pattern) sempre que aplicável para garantir flexibilidade.

## Como usar um componente

1. Navegue até o componente no menu lateral.
2. Acesse a guia de código (ou Code Viewer) para copiar a implementação.
3. Se o componente utilizar dependências externas (ex: Zustand, @base-ui/react), instale-as no seu projeto.
4. Cole os arquivos no seu projeto (como `modal.tsx`, `modal.module.css` e `theme.css`) e importe onde precisar.

Nenhuma configuração adicional pesada é necessária — cada funcionalidade da feature foi desenhada para ser copiada e funcionar.
